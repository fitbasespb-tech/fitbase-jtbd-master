**284 строки** (179 уникальных + 105 дубликатов) из следующих источников:

- **switch_other_systems** — 12 файлов, 104 строки. ⚠️ 10 из них = ОДНО интервью с Анной (танцы), 9 джобов × 10 копий = 90 дубликатов
- **switch_tables (Lime + другая танц.студия)** — 24 джоба
- **Unmanned (Speaker A)** — 5 джобов

### Агрегированный CSV
- `jobs_and_pains/jobs.csv` — 29 джобов с УТП Fitbase, без привязки к конкретному ЛПР

### Narrative-MD (только ссылки, требуют ручной обработки)
- UrbanFit (narrative-анализ)

## ⚠️ Важное про дубликаты

**Найдено 25 кластеров дубликатов = 105 строк-копий.**

Главное:
- **switch_other_systems** содержит 10 файлов с ОДНИМИ И ТЕМИ ЖЕ 9 джобами Анны (танцевальная студия). Имена файлов разные (`switch_from_1c`, `switch_from_bitrix`, `switch_mobifitness`...) но контент идентичный — это итерации одной обработки.
- **dmitry_network_** — 3 файла × 8 джобов = одно интервью Дмитрия в трёх вариантах обработки.

Что делать продакту:
- В Excel — **отфильтровать `Канон = 1`** на вкладке «Джобы (raw)» чтобы скрыть дубликаты
- Или открыть вкладку «Джобы (дедуп)» — там сразу только уникальные

## Структура колонок

| Колонка | Что |
|---------|-----|
| id | J001, J002… |
| source_file | путь к исходному файлу |
| source_type | csv_segment / csv_aggregated / md_jtbd / md_jtbd_narrative |
| interview_label | человекочитаемая метка интервью/ЛПР |
| business_segment | S1–S6 (Fitbase бизнес-сегменты) |
| job_segment | J1–J6 (джоб-сегменты по Замесину) |
| business_profile | профиль бизнеса (из метаданных CSV) |
| lpr_profile | профиль ЛПР |
| job_size | Big / Middle / Small |
| want_short | Хочу (кратко) |
| context_when | Когда (контекст, триггер) |
| want_result | Хочу получить (результат) |
| so_that | Чтобы (цель/эмоция) |
| importance | Важность 1–10 |
| satisfaction | Удовлетворённость 1–10 |
| gap | Gap (важность − удовлетворённость) |
| fitbase_value | Ценность Fitbase |
| previous_solution_problems | Проблемы прошлого решения |
| drivers | Драйверы выбора |
| barriers | Барьеры |
| lpr_quote | Цитата ЛПР |
| notes | Заметки |
| duplicate_group | D001…D025 — ID кластера дубликатов |
| is_canonical | 1 = каноническая, 0 = дубликат |
